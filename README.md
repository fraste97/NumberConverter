# NumberConverter

Simple program to convert numbers from a base (i.e. decimal) to another one (i.e. hexadecimal)

## How to use it

1. [Download it]()
1. Run it
  1. Insert the number
  1. Select the bases
  1. Click the "Converti" button 

## Requirements

* [Java 8 or latest](https://www.java.com/it/download/manual.jsp)
